Goal(s):
	- Easy design possibilities for users
	- Cross-platform, even for Mac
	- Qt enviorment

General ideas:
	- .ui files should be loaded at runtime to allow easy customization.
	- Documents are set up using .xml.
	- .xml files are stored for backup.
	- LaTeX support do deal with math formatting.
	- Possibilities to make plugins for manipulating text(Grammar check, etc).

Keywords:
	- QFormBuilder
	- QUiLoader
	- Dock Widgets


Design:
	- Aeres Office theme is to be light turquiose(Pale Turquiose) by default
	- Golden Section laws are to be taken into consideration
	- Something innovative and new
